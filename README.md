# 📖 3D Book Portfolio

## 🔗 Live Demo : https://3d-my-portfolio.netlify.app/

## 📌 Description

An interactive 3D portfolio website designed as a flipping book. Each page represents different sections like About Me, Skills, Projects, and Contact.

## 🚀 Features

* 3D book flip animation
* Interactive navigation between pages
* Smooth transitions and animations
* Responsive design

## 🛠️ Tech Stack

* HTML
* CSS (3D transforms & animations)
* JavaScript

## 🎨 Highlights

* Unique 3D UI design
* Creative way of presenting personal portfolio
* Focus on user experience and visual appeal


## 📁 Project Setup

1. Clone the repository
2. Open index.html in browser
